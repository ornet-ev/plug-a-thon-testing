## Project description

This project provides test resources used for running tests on SDPi Plug-a-thons. It helps in tracing the changes made to the test MDIB and test specification. It also serves as a database for test records.

The latest MDIB can be downloaded here: https://github.com/ornet-ev/plug-a-thon-testing/releases, the matching test specification is provided as a JSON file.
