# Definitions

## Discovery Proxy

A discovery proxy as specified in [SDPi](https://profiles.ihe.net/DEV/SDPi/index.html#_discovery_proxy)

## Reference Consumer

The service consumer implementation of a SOMDS Consumer as specified in [SDPi](https://profiles.ihe.net/DEV/SDPi/index.html#vol1_clause_sdpi_p_somds_consumer) that adheres to this test specification.

## Reference Provider

The service provider implementation of a SOMDS Provider as specified in [SDPi](https://profiles.ihe.net/DEV/SDPi/index.html#vol1_clause_sdpi_p_somds_provider) that adheres to this test specification.

## Localized Text Database

A test file that is shared between[ Reference Consumer](#reference-consumer)s and [Reference Provider](#reference-provider)s to verify certain properties of the texte provided by the [Reference Provider](#reference-provider)s localization service.
